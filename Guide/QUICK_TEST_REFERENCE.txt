================================================================================
TROUBLESHOOTING
================================================================================

Test not found?
  - Check spelling (case-sensitive!)
  - Use full class name: com.transitflow.service.DatasetServiceTest

Tests fail?
  - Clean and rebuild: .\mvnw.cmd clean compile test-compile
  - Run test again

Compilation errors?
  - Update dependencies: .\mvnw.cmd clean install -U

IDE not recognizing tests?
  - IntelliJ: File → Invalidate Caches → Restart
  - Eclipse: Project → Clean
